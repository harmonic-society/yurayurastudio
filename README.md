# Yura Yura STUDIO

千葉県で地域貢献できるWeb制作・集客支援のためのプロジェクト管理システム（ベータ版）。

## 特徴

- **プロジェクト管理**：進行状況の把握と共有（全ユーザーが閲覧可能）
- **リード案件管理**：案件化前の潜在案件の登録と議論（全ユーザーが登録・コメント可能）
- **チームコラボレーション**：クリエイター、営業、ディレクター間のスムーズな連携
- **ポートフォリオ管理**：制作物の記録と共有
- **ファイル共有**：プロジェクトファイルのアップロード・ダウンロード（全ユーザー利用可能）
- **ダッシュボード分析**：月別売上やクライアント別売上の可視化
- **モバイルアプリ対応**：AndroidアプリとしてCapacitorで実装

## 使い方ガイド

### アカウント登録・ログイン

1. **新規登録**: 初めてのユーザーは、トップページの「新規登録」ボタンからアカウント登録をリクエストできます。
   - 登録情報を入力し、送信すると管理者の承認待ちとなります。
   - 管理者が承認すると、登録したメールアドレスに通知が届きます。

2. **ログイン**: 登録が完了したら、ユーザー名とパスワードでログインできます。

### ダッシュボード

ログイン後、最初に表示されるダッシュボードでは以下の情報を確認できます：

- **進行中のプロジェクト**：現在進行中のプロジェクトを優先度順に表示
- **月別売上分析**：過去6ヶ月の売上推移をグラフで可視化（管理者のみ）
- **クライアント別売上**：上位5社の売上をグラフで表示（管理者のみ）
- **最近の通知**：プロジェクトの更新やメンションなどの通知
- **タイムライン**：チームメンバーの最新投稿

### プロジェクト管理

1. **プロジェクト一覧**: 「プロジェクト」メニューから、全てのプロジェクトを閲覧できます（全ユーザー）。
   - ステータス（未着手、進行中、完了、保留）でフィルタリングが可能です。
   - 検索機能で特定のプロジェクトを素早く見つけられます。
   - 報酬金額は管理者のみ閲覧可能です。

2. **プロジェクト詳細**: プロジェクト名をクリックすると、詳細画面が表示されます（全ユーザー閲覧可能）。
   - 基本情報（クライアント、期間、担当者など）
   - 担当者一覧
   - プロジェクト履歴
   - コメント・ディスカッション
   - プロジェクトファイル（アップロード・ダウンロード可能）
   - 成果物（ポートフォリオ登録）

3. **新規プロジェクト作成**: 全ユーザーが「新規プロジェクト」ボタンから、新しいプロジェクトを作成できます。
   - プロジェクト情報を入力
   - チームメンバーをアサイン
   - 納期、報酬、報酬分配ルールを設定

4. **プロジェクト編集**: 管理者は「編集」ボタンから情報を更新できます。一般ユーザーはステータスのみ変更可能です。

5. **プロジェクトファイル**: 全ユーザーがファイルをアップロード・ダウンロードできます。
   - ローカルファイルのアップロード
   - Google Driveファイルの紐付け
   - ファイル削除は管理者またはアップロード者のみ可能

### リード案件管理

1. **リード案件一覧**: 「リード案件」メニューから、案件化前の潜在案件を閲覧できます（全ユーザー）。
   - ステータス（新規、連絡済み、交渉中、有望、成約、失注）で進捗管理
   - 企業名、予算見込み、登録日で一覧表示

2. **リード案件詳細**: リード案件をクリックすると、詳細画面が表示されます。
   - 案件情報（企業名、説明、予算見込み、開始予定日）
   - 連絡先情報（担当者名、メールアドレス、電話番号）
   - コメント・ディスカッション（全ユーザーがコメント可能）

3. **新規リード登録**: 全ユーザーが「新規リード」ボタンから、潜在案件を登録できます。
   - タイトル、説明、企業名などの基本情報
   - 連絡先情報
   - 予算見込みと開始予定日
   - 担当者のアサイン
   - 獲得元の記録

4. **リード案件編集**: 全ユーザーがリード案件を編集できます。
   - ステータスの更新
   - 情報の追加・修正
   - コメントによる進捗共有

### ポートフォリオ

1. **ポートフォリオ一覧**: 「ポートフォリオ」メニューから、チーム全体の制作実績を閲覧できます。
   - URLベースの実績とファイルアップロードの両方に対応
   - プレビュー画像が自動生成されます

2. **ポートフォリオ追加**: 「新規追加」ボタンから、新しい制作実績を登録できます。
   - 案件名、説明文、制作種別
   - URLリンクまたはファイル（PDF、画像、Officeドキュメントなど）のアップロード
   - サムネイル画像の自動取得（URLの場合）または生成（ファイルの場合）
   - 公開/非公開設定

3. **ファイル管理**: 画像、PDF、その他のドキュメントをアップロードできます。
   - 画像はサムネイルとして表示され、クリックで拡大表示
   - PDFや他のファイルは適切なアイコンで表示され、ダウンロードまたは表示可能

4. **スキル管理**: 「スキル」タブから、自分のスキルセットを管理できます。
   - カテゴリ別にスキルを追加
   - 習熟度を設定

### チーム機能

1. **メッセージ**: 「メッセージ」メニューから、チームメンバーとプライベートメッセージを交換できます。
   - 1対1のダイレクトメッセージ
   - メッセージ検索機能
   - 未読通知

2. **タイムライン**: 「タイムライン」メニューで、チーム全体の活動を時系列で確認できます。
   - 新規投稿の作成
   - いいね・コメント機能
   - メンバーのアクティビティ確認

3. **チームメンバー**: 「チーム」メニューから、全メンバーのプロフィールを閲覧できます。
   - メンバーのスキルセット
   - 担当プロジェクト
   - 連絡先情報

### 通知システム

1. **アプリ内通知**: 画面上部の通知ベルアイコンで、未読通知を確認できます。
   - プロジェクトの更新
   - メッセージの受信
   - メンション通知
   - タスクのリマインダー

2. **メール通知**: 重要な通知はメールでも受け取れます。
   - 通知設定から受信する通知タイプをカスタマイズ可能

### 管理者機能

管理者ユーザーは以下の追加機能を利用できます：

1. **ユーザー管理**: ユーザーの追加、編集、停止などの管理が可能です。

2. **登録リクエスト管理**: 新規登録リクエストの承認・拒否を行えます。

3. **報酬配分設定**: プロジェクト完了時の報酬配分ルール（営業、ディレクター、クリエイター）を設定できます。

4. **売上分析**: ダッシュボードで月別売上とクライアント別売上を確認できます。

5. **プロジェクト編集**: 全プロジェクト情報の編集が可能です。

6. **報酬情報閲覧**: プロジェクトの報酬金額や分配状況を閲覧できます。

### プロフィール・設定

1. **プロフィール編集**: 「プロフィール」メニューから、自分のプロフィール情報を更新できます。
   - プロフィール画像
   - 自己紹介
   - 連絡先情報

2. **アカウント設定**: 「設定」メニューから、アカウント設定を変更できます。
   - パスワード変更
   - 通知設定
   - 表示言語設定
